MyProjects

ToDo List

A simple yet effective Java-based desktop application that helps users create, manage, and delete tasks efficiently. This project uses JDBC (Java Database Connectivity) to perform real-time CRUD operations (Create, Read, Update, Delete) on a MySQL database, ensuring data persistence and smooth task tracking.
Users can add due dates, priorities, and categorize tasks.
Data is stored securely in a relational database and dynamically fetched.
Exception handling and input validation ensure robustness and reliability.
Technologies Used: Java, JDBC, MySQL

AWS Cloud

A hands-on cloud infrastructure project demonstrating the creation and configuration of a custom Virtual Private Cloud (VPC) in AWS. This project includes the setup of public and private subnets, route tables, internet gateways, and NAT gateways to enable secure, scalable networking.
The architecture ensures secure communication between components and supports deployment of EC2 instances in both public and private zones. Security groups and Network ACLs were configured to enforce access control policies, ensuring isolation and secure traffic flow.
This project showcases foundational AWS networking skills and infrastructure as code practices using AWS Console and optionally, Terraform or AWS CLI.
Technologies Used: AWS VPC, EC2, IGW, NAT, Route Tables, Security Groups, IAM

Sentiment Analysis Model

An AI-powered machine learning project that analyzes customer sentiments from food reviews to determine public opinion on various dishes, restaurants, or food products. The goal was to classify reviews as positive, negative, or neutral, enabling better decision-making for businesses and improving customer experience.
The project uses Natural Language Processing (NLP) techniques for text preprocessing (tokenization, stopword removal, stemming) and vectorization (TF-IDF). A classification model (Logistic Regression, SVM, or Random Forest) was trained on labeled datasets, achieving high accuracy and performance.
The model was further evaluated using metrics like accuracy, precision, recall, and F1-score, and deployed in a user interface for real-time sentiment prediction on new inputs.
Technologies Used: Python, Scikit-learn, NLTK, Pandas, Matplotlib

Employee Burnout Prediction

This project focuses on identifying employees at risk of burnout using AI and machine learning. By analyzing various factors such as working hours, job role, stress indicators, satisfaction scores, and employee feedback, the system can predict burnout levels with high accuracy.
The goal is to provide HR teams with actionable insights to improve workplace wellness and employee retention.
Technologies Used:Python, Pandas, NumPy, Scikit-learn, Matplotlib
